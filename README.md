# CardMap

A simple iOS utility for credit card reward rules.

Save your cards, add notes, and instantly see which card to use for any spending category. Drag to rank your favorites.

## What it does

**My Cards** — Add your credit cards with optional notes. Drag to reorder by preference.

**Quick Reference** — Automatically shows the best card to use for each spending category, based on your saved cards and their reward rates. Drag to reorder categories. Supports rotating quarterly bonus categories (Chase Freedom Flex, Discover it Cash Back).

## What it doesn't do

No accounts. No tracking. No cloud. No financial data access. No spending analysis. No AI.

## Stack

- SwiftUI
- UserDefaults + Codable
- Bundled JSON for card reward rules

## Requirements

- iOS 17+
- Xcode 15+
